# Apple Services Optimizer

Find the most cost-effective way to subscribe to your favorite Apple services.

## Features

- Compare Apple One bundles vs individual subscriptions
- Account for carrier discounts (Verizon, T-Mobile)
- Student and corporate discount support
- Multi-currency support with automatic tax calculation
- Family Sharing optimization (1-6 members)
- Device promotion tracking

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation

1. Clone the repository
```bash
git clone https://github.com/ndlaroy99/apple-services-optimizer.git
cd apple-services-optimizer
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open http://localhost:5173 in your browser
